## Download Instructions

1. Download the latest executable release for your platform from the Releases section on GitHub.

2. Extract the ZIP file if necessary.

3. Run the executable file named:

- **CSV Merger.exe** (Windows)  
- **CSV Merger** (macOS/Linux)

---

## Usage

1. Launch the CSV Merger application.

2. Click the **Browse** button next to **Total CSV** to select your first CSV file.

3. Click the **Browse** button next to **Apple CSV** to select your second CSV file.

4. (Optional) Enter a delimiter in the **Delimiter** field if your CSV files use a specific delimiter (e.g., `,`, `;`, `|`, or `\t`). Leave blank to auto-detect.

5. Click the **Merge CSVs** button to start merging.

6. After the merge completes, the merged file will be saved as `merged_output.csv` in the same directory where you ran the application.

7. Check the status message at the bottom for confirmation or errors.
